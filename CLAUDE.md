Сеансы (index.html, главная) · Ретрит (retreat.html) · Обучение (training.html).

## Продукт и аудитория
Три направления:
1. **Сеансы** (главная) — индивидуальная работа с телом и внутренним состоянием.
   Три программы на выбор: «Атлант + крестец + спина» (1 сеанс), «Восстановление
   физического здоровья» (4–5 сеансов), «Комплексное восстановление тела и сознания»
   (5–6 сеансов, самая полная — визуально выделена карточка).
2. **Ретрит «Возвращение к себе»** — групповой формат, 3 часа, семь шагов
   (дыхательные практики → альфа-состояние → работа с подсознанием → интеграция).
3. **Обучение** — остео-биодинамический подход, 4 дня/40 часов, для специалистов
   телесных практик (массажисты, остеопаты, кинезиологи, косметологи).

Публичное имя everywhere: **Ержан Кабдрашит (Тусупов)**. Исключение — заголовок h1
на главной («Авторский метод Ержана Кабдрашита», без Тусупова — по тексту заказчика)
и строка под сертификатами (объясняет оба имени).

## Технический стек
- Три HTML-файла (index.html, retreat.html, training.html), общие `styles.css` и `app.js`
- Vanilla JS, без фреймворков; fluid-типографика через clamp()
- Общая шапка/футер повторены в каждом файле (нет server-side includes на GitHub Pages)
- Чистые URL без .html: nav/footer/canonical ссылаются на `./`, `./retreat`, `./training`
  (и на будущие `./oferta`, `./privacy`, `./consent`) — GitHub Pages отдаёт файл.html по
  запросу без расширения, но КАНОНИЧЕСКИЙ url должен быть без расширения (см. LANDING-GUIDE)
- Фото: photos/hero/ (hero.*, retreat.*, training.*), photos/portrait.*, photos/school/,
  photos/certs/ — WebP + JPG fallback
- Видео: video/method.mp4 + poster.jpg (на главной, блок «Ержан»)
- Деплой: GitHub Pages, репозиторий Xenaja/tusupov-method

## Запись — по городам, без общего WhatsApp/TG на всех
- **Казахстан → WhatsApp** (кнопки `data-wa="текст сообщения"`, номер `WA_KZ` в app.js)
- **Москва → Telegram-бот** (кнопки `data-tg="код_места_клика"`, `TG_BOT` в app.js,
  start-параметр = код места: program_hero_msk, program_dates, retreat_hero_msk,
  training_hero и т.д. — коды говорящие, свои на каждой странице)
- Кнопки записи и «узнать стоимость/даты» — ВСЕГДА пара (Москва + Казахстан) на страницах
  «Сеансы» и «Ретрит». На «Обучении» — ОДНА кнопка (только Telegram-бот, ветка обучения) —
  так задано в тексте, не пара.
- ⚠ ПЛЕЙСХОЛДЕРЫ в начале app.js: `TG_BOT="PLACEHOLDER_BOT"`, `WA_KZ="70000000000"`.
  Заменить перед запуском рекламы.
- Sticky-кнопка на каждой странице — одна, якорь `#zapis` (id стоит на первом
  cta-row в hero), скроллит к пиковым кнопкам, не открывает канал напрямую.

## Тире
В текстах — только среднее (–), не длинное (—).

## Юридические ограничения (важно!)
НЕЛЬЗЯ использовать слова: «лечение», «исцеление», «диагностика», «доктор»,
«целитель», «реабилитация», обещания «вылечим/избавим/гарантируем результат».
На странице «Обучение» отдельно смягчены две формулировки из исходных материалов:
«снятие боли и исцеление» → результаты для клиентов без мед. обещаний;
«постоянный поток клиентов» → «возможность практиковать сразу после обучения».
Стоимость нигде на сайте не публикуем — только через менеджера.
Деятельность: ИП Кабдрашит, Республика Казахстан.

## Решения, которые уже приняты
- Расписание указываем по Москве
- Противопоказания: онкология и эпилепсия — прямо названы в FAQ (страница «Сеансы»)
- Сертификаты оформлены на паспортное имя Ержан Кабдрашит — на сайте есть поясняющая приписка
- Счётчики на главной: 1800+ человек, 216 часов обучения (7 сертификатов: 6×32 + 24)
- Отзывы — скриншоты из Instagram; блок закомментирован на главной, ждём материалы
- Оферта/политика/согласие — заказчик пришлёт образцы; ссылки в футере уже стоят
  (./oferta ./privacy ./consent) и пока ведут на 404 — файлы создать, когда придут тексты
- Форм на сайте нет (только WhatsApp/Telegram-ссылки) — чекбоксы 152-ФЗ не требуются
- «Удостоверение о ПК + внесение в ФРДО» НЕ публикуем без подтверждённой лицензии —
  на странице «Обучение» стоит нейтральная формулировка + предупреждающая плашка
