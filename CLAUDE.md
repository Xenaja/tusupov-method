# Метод Ержана Тусупова — Landing

## Продукт и аудитория
Два направления:
1. Пятидневная программа работы с телом и внутренним состоянием (Москва, с 12 августа; также Алматы и Астана). Аудитория — люди с хроническим напряжением, болями, выгоранием, у которых массаж/процедуры не дали стойкого результата.
2. Школа метода — 5-дневное очное обучение для остеопатов, массажистов, кинезиологов, косметологов, телесных практиков. Ближайший поток — ноябрь, Казахстан. Стоимость на сайте НЕ публикуем — только через менеджера (репо публичное, цену здесь тоже не пишем).

## Цель лендинга
Основной канал — Telegram-бот (кнопки data-tg, start-параметр кодирует место клика:
program_menu / program_hero / program_dates / program_reviews / program_final /
program_price / program_sticky / program_footer / training). WhatsApp — запасной (кнопки data-wa,
два номера: программа и обучение).
⚠ ПЛЕЙСХОЛДЕРЫ в начале <script> в index.html: TG_BOT="PLACEHOLDER_BOT",
номера WA 70000000000 / 70000000001. Заменить перед запуском рекламы.
Тире в текстах — только среднее (–), не длинное.

## Стиль
Ощущение: спокойный, статусный, глубокий.
Фон: тёмный #141210 (одна светлая контраст-секция #EEE6D5).
Акцентный цвет: #C1A268 (золото).
Шрифт заголовков: Source Serif 4. Шрифт текста: Golos Text.
Деталь: зерно-оверлей, мотив трёх пересекающихся кругов (тело/эмоции/состояние) — он же favicon.

## Технический стек
- Один index.html (CSS + JS внутри), vanilla JS, без фреймворков
- Fluid-типографика через clamp(), breakpoint только 640px (полоса цифр)
- Фото: photos/hero/, photos/school/, photos/certs/ — WebP + JPG fallback
- Видео: video/method.mp4 (11 сек, H.264 720p) + poster.jpg
- Деплой: GitHub Pages, репозиторий Xenaja/tusupov-method

## Юридические ограничения (важно!)
НЕЛЬЗЯ использовать слова: «лечение», «исцеление», «диагностика», «доктор»,
«целитель», «реабилитация», обещания «вылечим/избавим/гарантируем результат».
Только нейтральные формулировки: работа с телом, восстановление, снятие напряжения,
гармонизация, телесная практика. Полный список замен — в info.docx (корень исходников).
Деятельность: ИП Кабдрашит, Республика Казахстан.

## Решения, которые уже приняты
- Цены на сайте не публикуем — «стоимость назовёт менеджер» (решение заказчика)
- Расписание указываем по Москве
- Противопоказания: онкология и эпилепсия — прямо названы в FAQ
- Сертификаты оформлены на паспортное имя Ержан Кабдрашит — на сайте есть поясняющая приписка
- Счётчики: 1800+ человек, 216 часов обучения (7 сертификатов: 6×32 + 24)
- Практикует с октября 2025, ~2000 человек принял
- Место жительства не указываем (возможно, Алматы)
- Отзывы — скриншоты из Instagram; блок закомментирован, ждём материалы
- Оферта/политика/согласие — заказчик пришлёт образцы; ссылки в футере закомментированы
- Форм на сайте нет (только WhatsApp-ссылки) — чекбоксы 152-ФЗ не требуются
